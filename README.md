# simple-calculator (시작점)

이 폴더는 **거의 비어 있습니다**. 핸드아웃을 따라가면서 한 단계씩 채워가게 됩니다.

## 폴더에 들어있는 것

- `README.md` (이 파일) — 시작 안내
- `.gitignore` — Git에서 무시할 파일 목록 (`node_modules/`, `coverage/` 등)

이게 전부입니다. 이 안에서 직접:

1. `npm init`으로 `package.json`을 만들고
2. `src/`에 HTML/CSS/JS를 작성하고
3. `tests/`에 테스트를 작성하고
4. `.github/workflows/`에 CI/CD 워크플로우를 작성하고
5. `Dockerfile`을 작성합니다.

## 시작하기

핸드아웃의 **STEP 0**부터 따라가세요.

```bash
node --version    # v20 이상이어야 합니다
git --version     # 설치되어 있어야 합니다
```

위 두 명령이 정상 출력되면 준비 완료입니다.
